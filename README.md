# llog

## Development Setup

1. Install dependencies

   ```bash
   bun i
   ```

2. Add environment variables

   ```bash
   # .env

   EXPO_PUBLIC_INSTANT_APP_ID=<ID>
   ```

3. Start the app

   ```bash
   bun dev
   ```
